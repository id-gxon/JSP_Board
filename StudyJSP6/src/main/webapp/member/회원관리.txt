회원관리.txt

 - 회원 가입
 	= 회원 정보 입력창에 사용자가 회원 정보 입력,
 	  입력한 회원 정보를 DB에 저장
 	  
 - 로그인 / 로그아웃
 	= 로그인 - 사용자의 아이디, 비밀번호를 입력,
 			   입력된 아이디, 비밀번호가 DB에 존재하는지 조회
 			   DB - 아이디 O: 회원
 			   		- 비밀번호 O: 본인(로그인 성공 - 로그인 아이디 정보를 session 영역에 저장)
 			   		- 비밀번호 X: 비밀번호 오류(로그인 실패)
 			      - 아이디 X: 비회원

 - 메인 페이지
 	= 로그인을 한 사용자만 접근 가능하도록 처리
 	  로그인을 하지 않은 경우 로그인 페이지로 이동
 	  
 	= 로그아웃 - 세션의 영역 객체 정보를 삭제
 	
 - 정보 조회
 	= 로그인을 성공한 사용자만 접근 가능
 	  DB에 접근해서 사용자 ID에 해당하는 모든 정보를 출력
 	  
 	  
 - 정보 수정
 	= 기존의 회원 정보를 보여준 다음 사용자의 변경 사항을 입력받는다.
 	  수정하기 버튼을 클릭하면 실제 데이터 베이스의 정보를 수정한다.
 	  
 - 회원 탈퇴
 	= 기존 비밀번호를 한 번 더 입력
 	  전달 받은 회원정보(id, pass)를 통해 실제 DB의 정보를 삭제
 	
 - 회원 목록(list) 조회
